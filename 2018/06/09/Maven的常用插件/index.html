Maven的常用插件

发表于 2018-06-09 | 分类于 后端 ， Maven |

Maven的核心仅仅定义了抽象的生命周期，具体的任务是交由插件完成，插件以独立的构件形式存在。Maven的生命周期与插件项目绑定，用以完成实际的构件任务。项目的构建实际上是在执行各个阶段时调用各自绑定的插件来具体完成构建步骤的。

一个Maven插件可以包含多个目标（Goal），每个目标完成一个任务。

Maven的插件有两种用法，一种是直接在命令行使用，比如mvn release:prepare 对应了maven-release-plugin执行prepare 目标 ，另一种是绑定到Maven的生命周期的某个阶段，比如maven-clean-plugin 绑定在clean生命周期的clean阶段，当执行mvn clean 时也就执行了。

Maven的内置插件

为了不用配置就能构建Maven项目，在Maven的生命周期中默认绑定了一些插件：

插件（plugin） 插件目标（goal） 生命周期（life cycle） 阶段（phase） 执行任务（task）
maven-clean-plugin clean clean clean 清理上一次构建的文件
maven-resources-plugin resources default process-resources 复制主资源文件至主输出目录
maven-resources-plugin testResources default process-test-resources 复制主资源文件至测试输出目录
maven-compiler-plugin compile default compile 编译主代码至主输出目录
maven-compiler-plugin testCompile default test-compile 编译主代码至测试输出目录
maven-surefire-plugin test default test 执行测试用例
maven-jar-plugin jar default package 创建项目jar包
maven-install-plugin install default install 将打包好的项目复制到本地仓库
maven-deploy-plugin deploy default deploy 将项目部署到远程仓库
maven-site-plugin site site site 建立站点
maven-site-plugin deploy site site-deploy 将站点部署到远程服务器

Maven的常用插件

maven插件在pom里的配置格式

maven-clean-plugin

默认绑定阶段： clean:clean

功能： 删除上一次构建生成的文件夹，它们分别是

  • project.build.directory

  • project.build.outputDirectory

  • project.build.testOutputDirectory

  • project.reporting.outputDirectory

目标 说明
clean:clean 清理上一次构建的文件
clean:help 显示帮助信息

同时删除自定义的文件

跳过清理

mvn clean -Dmaven.clean.skip=true

忽略错误

mvn clean -Dmaven.clean.failOnError=false

maven-compiler-plugin

默认绑定阶段： default:compile、default:test-compile

功能： 编译主源代码和测试源代码

包含目标：

目标 说明
compiler:compile 编译主源码
compiler:help 显示帮助信息
compiler:testCompile 编译测试源码

配置编译版本

或者直接可以

maven-resources-plugin

默认绑定阶段：default:process-resources、default:process-test-resources

功能： 处理主资源文件和测试资源文件

包含目标：

目标 说明
resources:copy-resources 复制插件属性配置的资源
resources:resources 把project.build.resources的资源文件复制到输出目录project.build.outputDirectory
resources:testResources 把project.build.resources的资源文件复制到输出目录project.build.testOutputDirectory
resources:help 显示帮助信息

maven-surefire-plugin

默认绑定阶段：default:test

功能： 可以使用TestNG和JUnit单元测试

  • **/*Test.java
  • **/*Tests.java
